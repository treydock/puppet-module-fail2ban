Overview

This module manages Fail2ban.

Usage

fail2ban

Install and configure fail2ban with SSH jail.

class { 'fail2ban':
  jails => ['sshd'],
}

Configure fail2ban to not ban a local subnet

class { 'fail2ban':
  jails            => ['sshd'],
  default_ignoreip => ['10.0.0.0/8'],
}
